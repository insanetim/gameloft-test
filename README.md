# Gameloft Test Task

A modern React + TypeScript application featuring an adaptive carousel for browsing products and a shopping cart for managing selected items.

This project is developed according to the technical specifications located in the `docs` folder.

## Demo

https://gameloft-test-insanetim.vercel.app

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode with Vite's dev server. Open [http://localhost:5173](http://localhost:5173) to view it in the browser. The page will automatically reload if you make changes.

### `npm run build`

Builds the app for production to the `dist` directory using Vite. The build is minified and the filenames include the hashes.
